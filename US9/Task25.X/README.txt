In order to run this file open MPLABX, click File >> Open Project select the folder Task24.X, then push run.
Connect SW pin of a quadrature encoder to PF4 then connect the CLk and DT to PF 2 or 3, but do not connect 2 ports to 1 pin,
connect a 5V power sourc to GND and +.
Then connect lights to all the pins in Pin bank D.
Lastly connect the Curiosity nano to a computer then flash the code.
